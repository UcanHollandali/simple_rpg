# SIMPLE RPG - Experiment Bank

## Purpose

This file stores speculative or not-yet-validated ideas.

It is intentionally non-authoritative.
Ideas here do not become project rules until promoted into an authoritative doc and logged in `DECISION_LOG.md`.

## Use Cases

Put ideas here when they are:
- interesting but not validated
- content-rich but not yet scoped
- mechanically possible but not yet proven
- likely to benefit from prototype testing first

## Current Experiment Buckets

### Combat Variants

- additional status families
- alternate brace rewards
- niche enemy behaviors
- advanced boss thresholds

### Item and Weapon Space

- extra weapon families
- more exotic consumables
- more specialized passive identities

### Progression Space

- additional build families
- run-start kit choices
- meta progression variants

### Map and Node Space

- richer event families
- more route condition types
- later-stage systemic twists

## Promotion Rule

An idea leaves this file only when:
1. it is judged worth keeping
2. it fits scope
3. its owning authoritative doc is updated
4. the decision is recorded if project-level
